중간점검

1. 함수란 무엇인가?

함수(function)는 특정 작업을수행하는명령어들의모음에이름을붙인것이다. 

2. 함수를 사용하는 이유는 무엇인가요?

함수를이용하면우리가여러번반복해야되는처리단계를하나로모아서
필요할때언제든지호출하여사용할수있다.

1. 함수에전달되는값을무엇이라고하는가?

인수 (argument)

함수를 호출할 때 넘겨주는 값

2. 함수 안에서 전달되는 값을 받는 변수는 무엇인가?

매개변수 (parameter)

함수 정의에서 값을 받는 변수

3. 사용자로부터 2개의 정수를 받아서 반환하는 함수

예시 코드:

def get_numbers():
    a = int(input("첫 번째 정수 입력: "))
    b = int(input("두 번째 정수 입력: "))
    return a, b

# 함수 호출
num1, num2 = get_numbers()
print(num1, num2)

1. 주어진사각형의면적을계산하는함수get_rect_area(w, h)를정의해보자. 여기서w는너비, h는높이이다. 

def get_rect_area(w, h):
    return w * h

2. main() 함수를 정의하고 main() 함수 안에서 get_rect_area()를 호출해보자.

def get_rect_area(w, h):
    return w * h

def main():
    width = int(input("너비 입력: "))
    height = int(input("높이 입력: "))
    
    area = get_rect_area(width, height)
    print("사각형의 면적:", area)

# main 함수 실행
main()

1. 인수와매개변수는다시한번설명해보자. 

인수 (argument)
함수를 호출할 때 전달하는 값

매개변수 (parameter)
함수 정의에서 값을 받는 변수

2. 디폴트인수란무엇인가? 예를들어보자. 

함수에서 기본값이 미리 정해진 매개변수

예시
def greet(name="홍길동"):
    print("안녕하세요", name)

greet()          # 기본값 사용
greet("영준")     # 전달값 사용

3. 키워드인수란무엇인가? 예를들어보자. 

매개변수 이름을 지정해서 전달하는 인수

def introduce(name, age):
    print(name, age)

introduce(name="영준", age=20)
introduce(age=20, name="영준")  # 순서 바꿔도 가능.

4. 매개 변수앞에* 기호가있다면무슨의미인가?

여러 개의 값을 튜플로 받는 것

1. 값을 반환하는키워드는무엇인가?

return

함수에서 결과값을 호출한 곳으로 돌려줄 때 사용

2. x와 y를 받아서x+y, x-y 값을 반환하는함수addsub(x, y)를 정의해보자.

x+y, x-y 두 값을 반환하는 함수

def addsub(x, y):
    return x + y, x - y

# 함수 호출
a, b = addsub(10, 5)

print(a)  # 합
print(b)  # 차

1. 지역 변수와전역변수는어떻게다른가?

지역변수는 함수 안에서 선언된 변수 해당 함수 안에서만 사용 가능

전역변수는 함수 밖에서 선언된 변수 프로그램 전체에서 사용 가능

2. 함수 안에서전역변수의값을읽을수있는가?

가능하다

3. 함수 안에서전역변수의값을변경하면어떻게되는가?

그대로는 변경되지 않는다
