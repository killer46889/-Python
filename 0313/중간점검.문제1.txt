중간점검 문제

1.변수를다시설명해보자.

a.다른언어의 변수는 상자와 같고 상자안에 값이 저장 되는 방식이라면 파이썬은 데이터가 객체 형태로 메모리에 저장되고 변수에는 객체를 참조할 수 있는 값이 저장된다.

3.변수이름을만들때지켜야하는규칙은무엇인가?

a.영문자, 숫자, _(언더바)만 사용할 수 있다
숫자로 시작할 수 없다
공백을 사용할 수 없다
대소문자를 구분한다
파이썬 예약어(키워드)는 사용할 수 없다

4.변수이름의첫번째글자로허용되는것은무엇인가?

a.영문자(a~z, A~Z) 또는 _(언더바)만 가능하다

5.파이썬에서고유한의미를가지고있는단어들을무엇이라고하는가?

a.키워드(keyword) 또는 예약어(reserved word) 라고 한다


1. 파이썬에는어떤자료형이있는가? 3가지만말해보자. 

a.정수형 (int)
실수형 (float)
문자열 (str)

2. 파이썬변수에정수를저장하였다가실수를저장하는것도가능한가? 그이
유는무엇인가?

a.가능하다.
이유는 파이썬은 동적 타입 언어이기 때문에 변수의 자료형이 자동으로 바뀌기 때문이다.

3. 변수의자료형을알려면어떤함수를사용하는가?

a.type() 함수를 사용한다.

1. 복합 대입연산자 x *= y의 의미를 설명하라.

x = x * y 와 같은 의미이다.
즉, x에 y를 곱한 결과를 다시 x에 저장한다.

2. 10 % 6의 값은 무엇인가?

4
(나머지를 구하는 연산이므로 10 ÷ 6의 나머지)


3. 나눗셈 연산인 10 // 6의 값은 얼마인가?
1
(몫만 구하는 정수 나눗셈)


4. 다음의 할당문에서 무엇이 잘못되었는가?

3 = x

→ 숫자(상수)에는 값을 저장할 수 없다.
→ 변수는 왼쪽에 와야 하므로 x = 3이 맞다.

5. 다음의 할당문이 실행된 후의 변수 a, b, c의 값은?

a = b = c = 100

 a = 100, b = 100, c = 100
(세 변수 모두 같은 값이 저장된다)
